Linguagens e Tecnologias

Linguagens que eu estudo e desenvolvo meus projetos

HTML

HTML é uma linguagem de marcação que usa tags para estruturar conteúdo na web, permitindo criar páginas com textos, imagens, links e outros elementos que os navegadores podem interpretar e exibir.

CSS

CSS é uma linguagem de estilização que controla a aparência visual de páginas web, definindo cores, layouts, fontes e outros aspectos de design dos elementos HTML.

JavaScript

JavaScript é uma linguagem de programação que adiciona interatividade e dinamismo às páginas web, permitindo manipular elementos, responder a eventos do usuário e atualizar conteúdo em tempo real.

PHP

PHP é uma linguagem de programação voltada para desenvolvimento web do lado do servidor, que permite criar páginas dinâmicas e interagir com bancos de dados.

SQL

SQL é uma linguagem para gerenciar e manipular bancos de dados relacionais, permitindo criar, consultar, atualizar e excluir dados de forma estruturada.

Tailwind

Tailwind é um framework CSS que utiliza classes utilitárias pré-definidas para estilizar elementos HTML diretamente no markup, permitindo um desenvolvimento mais rápido e consistente sem escrever CSS personalizado.

C#

C# é uma linguagem de programação orientada a objetos desenvolvida pela Microsoft, usada principalmente para criar aplicações Windows, jogos com Unity e sistemas web usando .NET Framework.

Java

Java é uma linguagem de programação orientada a objetos, multiplataforma e amplamente utilizada para desenvolver aplicações empresariais, Android, sistemas web e programas desktop.

Ferramentas

Ferramentas que eu utilizo no dia a dia

Visual Studio Code

O Visual Studio Code é um editor de código-fonte gratuito e altamente personalizável, que oferece recursos como realce de sintaxe, depuração, integração com Git e uma vasta biblioteca de extensões para diferentes linguagens de programação.

Visual Studio

O Visual Studio é um ambiente de desenvolvimento integrado (IDE) robusto da Microsoft, focado principalmente em desenvolvimento .NET, oferecendo ferramentas avançadas para codificação, depuração, teste e implantação de aplicações.

GIT

Git é um sistema de controle de versão que permite rastrear e gerenciar mudanças no código-fonte, facilitando o trabalho colaborativo entre desenvolvedores e o versionamento de projetos.

NetBeans

O NetBeans é um ambiente de desenvolvimento integrado (IDE) gratuito e de código aberto que oferece uma plataforma completa para desenvolvedores criarem aplicações em diversas linguagens de programação, como Java, JavaScript, HTML5, PHP, C/C++ e outras.

PostgreSQL

O PostgreSQL é um sistema gerenciador de banco de dados relacional (SGBD) de código aberto, amplamente utilizado e reconhecido por sua robustez, confiabilidade e flexibilidade. Desenvolvido por uma comunidade global, o PostgreSQL oferece um conjunto completo de recursos para o gerenciamento de dados.

PHPMyAdmin

O phpMyAdmin é um aplicativo web gratuito e de código aberto que facilita a administração de bancos de dados MySQL e MariaDB. Através de uma interface intuitiva, você pode realizar diversas tarefas sem a necessidade de escrever comandos SQL complexos.
